# Sitio Web Estático de Laurizos

## Descripción

Este proyecto es un sitio web estático para Laurizos, una marca especializada en productos y accesorios para cabello rizado. El sitio web ofrece una interfaz limpia y fácil de usar, mostrando diversos productos y facilitando la navegación para los usuarios.

## Estructura del Proyecto

El proyecto está compuesto por los siguientes archivos y directorios:

```md

laurizos-static
├── public
│   ├─
├── index.html
└── README.md
```

## Instrucciones de Instalación

1. Clona el repositorio o descarga los archivos del proyecto.
2. Asegúrate de tener un entorno de servidor local configurado (por ejemplo, XAMPP, WAMP).
3. Coloca la carpeta `laurizos-static` en el directorio correspondiente de tu servidor (por ejemplo, `htdocs` para XAMPP).
4. Abre tu navegador y accede a `http://localhost/laurizos-static/index.html` para ver el sitio web.

## Uso

- Navega por el sitio web usando el encabezado y la navegación de categorías.
- Utiliza la barra de búsqueda para encontrar productos específicos.
- Explora los productos destacados en la sección principal.
- Visualiza los detalles de los productos en la cuadrícula de productos.
- Accede a las funciones de cuenta de usuario y favoritos desde el encabezado.
